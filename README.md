# segMod, MIT license

Welcome!  This project contains a mixture of reverse-engineered documentation, Python and C code for use with Segway Ninebot 'x3 series' scooters.  Developed on GT3 Pro and F3 models, using ESP32 and Raspberry Pi (or other small Linux), it should also apply to G3 and ZT3.
see https://github.com/MacintoshKeyboardHacking/segMod/wiki for resources

myBLEv4 is the current focus, a small ESP32s3 module is used to intercept communications between the BLE and VCU; this allows continued use of the stock display with new WiFi capabilities

myBLEv2 was the previous beta which removed and replaced the BLE module entirely with a limited WS2812 display

As always work is ongoing; for discussion and demonstration see my youtube https://www.youtube.com/@MacintoshKeyboardHacking/streams
